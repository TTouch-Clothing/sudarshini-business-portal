# WooCommerce MERN Dashboard

Starter full-stack project for syncing WooCommerce orders into a MERN dashboard.

## Included
- Frontend: React + Vite + React Router + React Hot Toast + React Loading Skeleton + Lucide icons
- Backend: Node.js + Express + MongoDB + Mongoose + JWT auth + Cloudinary support
- WooCommerce plugin PHP file for order sync + Discord notify

## Run
### Backend
cd backend
cp .env.example .env
npm install
npm run dev

### Frontend
cd frontend
cp .env.example .env
npm install
npm run dev

## Notes
- This is a production-style starter scaffold with the main architecture and key pages/components.
- Replace demo secrets, webhook URL, and environment values before production.
